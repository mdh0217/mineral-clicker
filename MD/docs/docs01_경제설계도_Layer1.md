# ⛏️ 광물 키우기 (가제) — Layer 1: 경제 설계도

> 이 문서는 게임의 **모든 재화 흐름**을 정의하는 최하단 설계 문서다.
> 이후 모든 시스템은 이 경제 위에 얹힌다. **설계가 확정되기 전에는 코드를 짜지 않는다.**

| 항목 | 내용 |
|---|---|
| 버전 | v1.2 (오프라인 확장 및 소과금 타깃 모델 통합본) |
| 범위 | Layer 1 — 경제 *구조*. 정확한 수식은 Layer 2에서 확정 |
| 한 줄 정의 | 온라인 플레이로 도파민을 얻고, 오프라인은 4시간 확장형 저축 시스템과 3-Way 부스팅으로 보완하는 방치형 클리커 |

## 1. 코어 루프 — 한 장 요약

```text
[온라인 루프: 적극 유치]
 광물 클릭 ─▶ 원석 획득 ─▶ 확률형 대박(Jackpot) 및 특수 재화 가동 ─▶ 골드 / 강화석 폭발
            골드 + 강화석 ─▶ [검 강화] ─▶ 디펜스 모드 스펙 상승 (성공률 중심)
            강화 실패 시 ─▶ '부서짐' 패널티 ─▶ 골드/젬으로 복구 유도

[오프라인 루프: 보존 및 소과금 연동]
 게임 종료 ─▶ 기본 4시간(업그레이드로 확장 가능) 동안 '기본 재화만' 누적 (대박 0%)
            접속 시 3-Way UI ─┬─▶ 1. 기본 수령 (100%)
                              ├─▶ 2. 광고 시청 ─▶ @배 수령 (광고 제거 패스 구매 시 즉시 활성화)
                              └─▶ 3. 젬 소모 ─▶ @배 수령
```

## 2. 재화 정의 (Currency Bible)

| 재화 | 티어 | 희소성 | 역할 |
|---|---|---|---|
| **골드** (Gold) | 1 | 풍부 | 범용 소프트 재화. 강화 수수료·상점·오프라인 시간 확장 업그레이드 비용 |
| **원석** (Ore, 종류별) | 1 | 풍부~보통 | 채굴 직접 산출물. 판매로 골드 전환. 상위 등급은 온라인에서만 수급 |
| **강화석** (Enhance Stone) | 2 | 보통(살짝 부족) | 검 강화 시도당 소모. 디펜스 모드 및 원석 분해로 수급 |
| **속성 정수** (Essence) | 2 | 부족 | 검 인챈트 부여/재설정. 온라인 플레이 전용 던전 보상 |
| **보호권** (Protect Scroll) | 2 | 희귀 | 강화 실패 시 부서짐 패널티 방지. 상점 구매 및 조각 합성 |
| **보호권 조각** | 2 | 보통 | 일정 개수 모아 보호권 합성. 온라인 디펜스 고층 및 귀족 광산 임대 보상 |
| **고대 광석** | 2 | 보통 | [온라인 버프 전용] 환생 상점에서 온라인 플레이 효율 극대화 연구 재료로 소모 |
| **환생 포인트** (Rebirth Pt) | 3 | 누적형 | 환생 시 획득. 영구 배율 스킬트리 및 오프라인 시간 최대치 확장 |
| **젬** (Gem) | 3 | 프리미엄 | 소과금 전용. 오프라인 @배 수령, 검 부서짐 즉시 복구, 보호권 직접 구매 |

## 3. Source / Sink 매트릭스 ★

| 재화 | 들어옴 (Source) | 빠짐 (Sink) | 균형 장치 |
|---|---|---|---|
| 골드 | 원석 판매, 디펜스 처치, 오프라인 보상 | 강화 수수료, 검 부서짐 복구비, **오프라인 시간 업그레이드** | 후반 가치 희석을 시간 확장 비용으로 소각 |
| 원석 | 광물 클릭, 오프라인 자동 채굴 | 판매(→골드), 강화 재료 | 오프라인 시 일반 등급만, 온라인 시 희귀 등급 산출 |
| 강화석 | 디펜스 웨이브, 던전, 원석 분해 | 검 강화 시도당 소모 | 항상 약간 부족하게 유지하여 온라인 플레이 유도 |
| 보호권 | 상점 직접 구매(젬), 조각 합성 | 강화 시 패널티 방지 1회 소모 | 과금러도 무한 구매 불가능하게 일일 상점 수량 제한 |
| 보호권 조각 | 귀족 광산 임대 버프(드롭률 3배), 디펜스 고층 | 보호권 제작 결합 | 무과금러도 노가다로 보호권을 얻을 수 있는 통로 |
| 고대 광석 | 귀족 광산 임대 시 추가 드롭, 온라인 전용 파밍 | 환생 전역 버프 연구 연구비 | 온라인 켜놓기 효율을 올리는 연구 전용으로 Sink 도메인 한정 |
| 환생 포인트 | 환생 실행 (누적 채굴량 기반) | 영구 배율 스킬, 오프라인 시간 확장 | 환생 사이클을 돌릴 강력한 메타 목표 제공 |
| 젬 | 소과금 결제, 일일 미션 및 업적 무료 제공 | 오프라인 @배 즉시 수령, 부서짐 즉시 수리, 보호권 구매 | 시간을 돈으로 사는 콤팩트한 편의성 도메인 제한 |

## 4. 경제 건전성 및 리스크 방어 장치

1. **검 강화 실패 패널티 방어벽:** 실패 시 검이 파괴되어 데이터가 날아가는 무자비한 방식 대신 성능이 50%로 다운되는 '부서짐' 상태로 전환된다. 복구를 위해 골드나 소량의 젬을 쓰게 만들어 재화 순환(Sink)을 유도하되 유저가 성장이 막혀 접는 사태를 원천 차단한다.
2. **오프라인 시간 저장 한계선(Cap) 제공:** 방치 시스템의 기본 한계치를 **4시간**으로 타이트하게 묶어 초반 인플레이션을 제어하고, 이를 확장하는 것 자체를 중반부 핵심 성장 콘텐츠(골드 및 환생 포인트 소모처)로 활용한다.
3. **완전 난수(Pure RNG)의 표면화:** 시스템 내부적으로는 통제용 알고리즘이 작동하더라도 유저 화면에는 철저히 "확률적으로 럭키 스트레이크가 터지는 감각"만 제공하여 클리커 특유의 날것의 도파민을 보존한다.
4. **광고 제거 패스의 최상위 가치화:** 오프라인 3-Way 보상 중 '광고 보고 @배 받기'의 피로도를 영구 제거해 주는 패스를 제공함으로써 소과금 유저층의 고정적인 첫 결제를 유도하고 리텐션을 확보한다.
